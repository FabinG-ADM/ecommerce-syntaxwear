# SyntaxWear - E-commerce de Sneakers

SyntaxWear é um projeto de e-commerce fictício focado na venda de tênis e sneakers. O site apresenta uma interface moderna, responsiva e focada na experiência do usuário, destacando coleções exclusivas e categorias diversas.

![SyntaxWear Preview](./images/banners/hero.jpg)

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada com o uso de variáveis (Custom Properties), Flexbox e CSS Grid.
- **Design Responsivo**: Interface totalmente adaptada para dispositivos móveis e desktops.

## 📦 Estrutura do Projeto

A organização dos arquivos segue uma estrutura modular para facilitar a manutenção:

```text
├── css/
│   ├── componentes/     # Estilos modulares (header, footer, hero, etc.)
│   ├── base.css         # Estilos base e tipografia
│   ├── reset.css        # Reset de estilos padrão para consistência entre navegadores
│   └── variables.css    # Definição de paleta de cores e variáveis globais
├── images/
│   ├── banners/         # Imagens de destaque e backgrounds
│   ├── icons/           # Ícones em formato SVG
│   ├── logo/            # Logotipo da SyntaxWear
│   └── products/        # Galeria de imagens dos produtos
└── index.html           # Ponto de entrada principal da aplicação
```

## ✨ Funcionalidades

- **Navegação Inteligente**: Menu superior com categorias e acesso rápido à conta, ajuda e carrinho.
- **Menu Mobile**: Implementado com técnica de checkbox para navegação em telas menores sem dependência de JavaScript.
- **Seção Hero Interativa**: Destaque principal com chamadas para ação (Call to Action).
- **Categorias de Estilo**: Navegação visual por estilos (Casual, Esporte, Moderno, Futurista).
- **Grade de Produtos**: Layout dinâmico utilizando CSS Grid para exibição de coleções.
- **Rodapé Institucional**: Newsletter, redes sociais e links de navegação estruturados.

## 🛠️ Como executar o projeto

Como este é um projeto estático (HTML/CSS), não é necessária nenhuma instalação complexa:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/ecommerce-syntaxwear.git
   ```
2. Navegue até o diretório:
   ```bash
   cd ecommerce-syntaxwear
   ```
3. Abra o arquivo `index.html` em qualquer navegador web moderno.

*Dica: Se estiver usando o VS Code, utilize a extensão **Live Server** para visualizar as alterações em tempo real.*
